diglib
======

A (personal) digital document management software.

Requirements (mostly Debian/Ubuntu packages):

* djvulibre-bin
* ghostscript
* imagemagick
* poppler-utils
* python-cairo
* python-djvu
* python-imaging
* python-magic
* python-poppler
* python-sqlalchemy
* python-xapian
* ssdeep for Python (http://bitbucket.org/denis/ssdeep)
