diglib
======

A personal digital document management software.

Requirements
------------

Most requirements can be installed as Debian/Ubuntu packages.

* djvulibre-bin
* ghostscript
* imagemagick
* poppler-utils
* python-cairo
* python-djvu
* python-imaging
* python-magic
* python-poppler
* python-sqlalchemy
* python-xapian
* [ssdeep](https://pypi.python.org/pypi/ssdeep)
