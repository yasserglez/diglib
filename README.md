diglib
======

A (personal) digital document management software.

Requirements (mostly Debian/Ubuntu packages):

* python-poppler
* poppler-utils
* python-cairo
* python-magic
* python-sqlalchemy
* python-xapian
* python-imaging
* python-djvu
* djvulibre-bin
* ssdeep for Python (http://bitbucket.org/denis/ssdeep)
* imagemagick
* ghostscript
